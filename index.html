color: Propiedad Indica el color del texto de un elemento. El color se indica con el nombre del color en inglés, con código hexadecimal o con código RGB. El color se hereda. Son dos valores para cada color: 2 para rojo, 2 para verde y 2 para azul. En hexadecimal, blanco es todas F (máximo valor) y negro es todos 0 (mínimo valor). En estos dos casos, se puede poner #FFF (en lugar de #FFFFFF) o #000 (en lugar de #000000). h1 {color: red;} Clase 1 Lunes 05 de Agosto
h1, p, etc. Selector Selecciona el elemento (puede ser cualquiera de los elementos, no solo h1 o p) para darle una clase, en CSS o dentro de la etiqueta style en HTML. <style type="text/css"> h1{color: red;} p { color: blue;} </style> Clase 1 Lunes 05 de Agosto

id="" Atributo Nombra un id y le da atributos. El nombre puede ser cualquiera. Se indica que es un id anteponiendo el símbolo de numeral al nombre. <style type="text/css"> #titulo {color: orange;} </style> Clase 1 Lunes 05 de Agosto
